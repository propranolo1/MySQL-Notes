MySQL命令

## SQL的分类

### DDL 数据定义语言

CREATE ALTER DROP RENAME TRUNCATE

### DML 数据操作语言

INSERT DELETE UNDATE SELECT

### DCL 数据控制语言

COMMIT ROLLBACK SAVEPOINT GRANT REVOKE

## 查看数据库

```mysql
show databases;
```

## 使用数据库

```mysql
use 数据库名称;
```
